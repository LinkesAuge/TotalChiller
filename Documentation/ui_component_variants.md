# UI Component Variants And States

This document defines the core UI components, their variants, and recommended states for the Total Chillers theme.

## Buttons
### Variants
- Primary: gold gradient, elevated, used for main actions.
- Secondary: dark surface with gold outline.
- Danger: red accent for destructive actions.

### States
- Default
- Hover (brighter gold, slight lift)
- Active (pressed, reduced shadow)
- Disabled (muted text, no glow)

## Badges
### Variants
- Rank: gold rim, uppercase
- Status: validated, warning, error
- Info: neutral info tag

## Tabs
### Variants
- Standard pill tabs
- Compact tabs for table filters

### States
- Active (gold stroke + glow)
- Hover (lighter text)
- Disabled (muted)

## Cards
### Variants
- Standard panel (glass surface)
- Emphasis panel (gold top glow)
- List item card (compact)

## Tables
### Variants
- Parchment table
- Dark table (optional for admin)

### States
- No hover effects on container rows (hover only on buttons/links)
- Selected rows (light gold overlay)
- Validation status cells (valid/invalid)
- Correction status cells (corrected highlight)

## Forms
### Inputs
- Text input
- Dropdown
- Date input
- Search field

### States
- Default
- Focus (gold ring)
- Error (red border)
- Disabled (muted)

## Alerts
### Variants
- Info (blue)
- Success (green)
- Warning (amber)
- Error (red)
